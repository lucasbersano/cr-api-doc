    Introdução

    Bem-vindo a documentação do Atendimento Total. API é baseada no padrão Graphql, é possível explorar utilizando graphiql em https://apitest.crhoteisbrasil.com.br/api. Para os exemplos utilizamos a lib "graph-request".

    Todas as requisições precisam de uma chave para pode interagir com API. As resposta são em JSON, caso não tenha experiência com GraphQL lei esta documentação. https://graphql.org/learn/queries/

    Autenticação

    Para qualquer query, precisa anexar a chave API.

    import { request } from "graphql-request";

const query = `{
    acomodacoesComTarifario(
      acomodacaoComTarifarioSelect: { chave: "sua-chave" }
    ) {
      maxadt
      msgadtes
    }
}`;

request("https://apitest.crhoteisbrasil.com.br/api", query).then(data =>
  console.log(data)
);

    Lembrese de trocar sua-chave para sua chave de accesso

    A autenticação é feita da seguinte maneira, dentro dos argumentos do query, deve se passar o chave. Contate seu administrador para ter acceso a chave.

    Lembre de trocar sua-chave para sua chave de acesso

    Hoteis

    Listar Hoteis

    Liste todo os hoteis disponiveis e seu codigos. Passe um codhotel no argumentos para filtra por um hotel específico.

    import { request } from "graphql-request";

const query = `{
  usuarios(usuarioSelect: {chave: "sua-chave"}) {
    codhotel
    nomehotel
    endereco
    cidade
  }}
}`;

    O comando de cima retornar o seguinte JSON

    {
  "data": {
    "usuarios": [
      {
        "codhotel": "0777",
        "endereco": "Av. Beira Mar, 777",
        "nomehotel": "Aquaville Hotel",
        "cidade": "Porto Seguro"
      },
      {
        "codhotel": "0019",
        "nomehotel": "Beach Mar Resort",
        "endereco": "Rua Erivam França, 172",
        "cidade": "Natal"
      }
    ]
  }
}

    Argumentos

    Argumento Tipo Descrição
    top Int Limita a quantidade de resultados
    chave String Chave API
    codhotel String Código do hotel, passe o codigo para filtrar

    Parâmentros de retorno

    Parâmetro Tipo Descrição
    codhotel String Código do hotel
    nomehotel String Nome do hotel
    endereco String Endereço do hotel
    bairro String Bairro
    cidade String Cidade
    estado String Estado
    cep Int CEP
    cnpj String CNPJ
    fone String Fone
